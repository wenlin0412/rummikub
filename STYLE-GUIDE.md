# Low-Poly 暖色系桌遊 UI 風格規格

> 一套以「低面數幾何（Low-Poly）」為核心的暖色調桌遊視覺語言。
> 適用於拉密、麻將、撲克、卡牌對戰等桌遊類 Web / Mobile 遊戲。
> 本文件僅描述視覺風格與 UI 規則，不包含程式實作。

---

## 一、設計理念

| 關鍵字 | 說明 |
|--------|------|
| **Low-Poly 立體感** | 所有元素以「偏移陰影」模擬幾何切面的立體紙模效果，不使用柔和漸層陰影 |
| **暖色奶油系** | 背景與填充以奶油色、蜜桃色為主調，營造溫暖親切的手感 |
| **粗體圓潤** | 字體全域使用粗黑體（700/900），圓角搭配厚邊框，風格年輕活潑 |
| **觸感回饋** | 按鈕按下時「陷入」、牌面 hover 時「浮起」，所有互動都有物理直覺 |

## 二、色彩系統

### 2.1 基底色（背景 / 填充）

| 名稱 | 色碼 | 用途 |
|------|------|------|
| Warm | `#FFF0E1` | 全域頁面背景底色 |
| Peach | `#FFDCC5` | 選中狀態的填充、幾何背景花紋色 |
| Cream | `#FFF8F0` | 卡片內填充、輸入框背景、牌面底色 |
| White | `#FFFFFF` 或 `white/90` | 面板、牌面、header 背景（帶半透明） |

### 2.2 主題色（品牌 / 強調）

| 名稱 | 色碼 | 用途 |
|------|------|------|
| Orange | `#FF9B5E` | 主品牌色、主按鈕、標題、邊框強調 |
| Yellow | `#FFD166` | 次要強調、計分板、徽章底色 |
| Green | `#7ECE8A` | 成功、確認、「輪到你」、選取高亮 |
| Pink | `#FFB5C2` | 裝飾、對手手牌數徽章 |
| Blue | `#89C4E1` | 輔助資訊、排序按鈕、借牌標記 |

### 2.3 文字色

| 名稱 | 色碼 | 用途 |
|------|------|------|
| Dark | `#5C4033` | 主要文字色（深咖啡棕） |
| Light | `#A0826D` | 次要文字、label、提示文字 |

### 2.4 遊戲物件色（牌面數字 / 標記）

| 名稱 | 色碼 | 用途 |
|------|------|------|
| Tile Red | `#E8564A` | 紅色牌數字、錯誤提示、撤銷按鈕 |
| Tile Blue | `#4A8FE8` | 藍色牌數字 |
| Tile Yellow | `#E8B84A` | 黃色牌數字 |
| Tile Black | `#3A3A3A` | 黑色牌數字 |

### 2.5 色彩使用原則

- 背景永遠使用基底色，不使用純白或純黑
- 邊框色 = 主題色 + 透明度（如 `orange/20`、`orange/30`）
- 按鈕選中態 = 主題色填滿 + 白色文字
- 按鈕未選中態 = 透明底 + 主題色邊框 + 淺色文字
- 錯誤/警告狀態使用 Tile Red 搭配 10% 透明度底色

### 3.1 字體設定

| 層級 | 字體 | 字重 |

| 備援 | sans-serif | — |

### 3.2 文字尺寸規範

| 用途 | 手機尺寸 | 桌機尺寸 |
|------|----------|----------|
| 大標題（遊戲名稱） | 2.25rem (text-4xl) | 3rem (text-5xl) |

| 按鈕文字 | 0.75rem (text-xs) | 0.75rem |

- 副標題使用 Light 色 + Bold
- 開啟 `antialiased` 抗鋸齒

## 四、Low-Poly 立體效果系統

這是本風格的核心特色。所有立體感透過**偏移實心陰影**（不是 blur shadow）實現。

### 4.1 陰影層級

| 層級 | 陰影值 | 適用元素 |
|------|--------|----------|
| 大陰影 | `6px 6px 0px rgba(92,64,51,0.15)` | 主面板、Modal、大卡片 |
| 小陰影 | `3px 3px 0px rgba(92,64,51,0.15)` | 牌面、小徽章、小卡片 |
| 牌面陰影 | `3px 3px 0px rgba(92,64,51,0.12)` | 各種牌面元素 |
| 選取陰影 | `5px 5px 0px rgba(126,206,138,0.6)` | 被選中的牌（綠色陰影） |

> 陰影色固定使用深咖啡棕（Dark 色）的低透明度版本，維持統一調性。

### 4.2 邊框厚度系統

「底邊 + 右邊加粗」是立體感的關鍵，模擬「從左上方光源照射的紙模」。

| 元素類型 | 上/左邊框 | 下/右邊框 | 效果 |
|----------|-----------|-----------|------|
| 大面板 | 2px | 8px | 強烈立體，用於主卡片、Modal |
| 按鈕 | 2px | 4px | 中等立體，用於所有可點擊按鈕 |
| 小元素 | 2px | 3px | 微立體，用於徽章、小按鈕、header 按鈕 |
| 牌面 | 2px | 4px | 牌面立體感 |
| 輸入框 | 2px | 4px | 同按鈕層級 |

邊框色根據語境選擇：
- 一般狀態：主題色 + 20%~30% 透明度
- 選中狀態：主題色 100%
- 按下狀態：邊框消失（模擬被按入）

### 4.3 按鈕按壓效果

按鈕按下（`:active`）時模擬「被壓入平面」：

1. 元素向右下平移 4px（`translate(4px, 4px)`）
2. 底邊框與右邊框寬度歸零
3. 過渡時間 0.1 秒

視覺效果：按鈕像一塊凸出的紙板被按平了。

## 五、元件規格

### 5.1 牌面（Tile）

牌是桌遊 UI 中最核心的元件。

| 屬性 | 手牌 | 桌面牌 |
|------|------|--------|
| 尺寸（手機） | 34 × 48 px | 34 × 46 px |
| 尺寸（桌機） | 42 × 58 px | 40 × 56 px |
| 圓角 | 4px (rounded) | 4px |
| 底色 | 白色 | 白色 |
| 邊框 | 2px 灰色，底/右 4px | 同左 |

#### 牌面狀態

| 狀態 | 視覺變化 |
|------|----------|
| **正常** | 小陰影，靜止 |
| **Hover** | 向左上偏移 2px，陰影加深至 5px |
| **選取** | 綠色邊框 + 綠色陰影，向左上偏移 3px |
| **剛抽到** | 綠色光暈脈動 2 秒後恢復 |
| **借用中** | 藍色邊框 + 藍色光暈 + 右上角圓形「↩」標記 |
| **本回合打出** | 綠色發光 + 左上角「✦」星號標記 |
| **無效牌組中** | 所在牌組背景變為紅色 10% + 紅色邊框 |

#### 牌面內容排版

- 數字牌：上方大號數字 + 下方小圓點（`●` 重複，最多 4 個）
- 特殊牌：居中 SVG 圖示 + 下方小號中文名稱標籤
- 數字顏色對應牌色（紅/藍/黃/黑）

### 5.2 按鈕

| 類型 | 底色 | 文字色 | 邊框色 | 用途 |
|------|------|--------|--------|------|
| 主要行動 | Orange 漸層 | 白色 | 深 Orange | 開始遊戲、加入房間 |
| 成功/確認 | Green | 白色 | 深 Green | 開始遊戲、結束回合 |
| 一般操作 | Cream | Dark | Orange/30 | 出牌、加入、取牌等 |
| 危險操作 | Cream | Tile Red | Red/30 | 撤銷 |
| 輔助/次要 | Blue/10 | Blue | Blue/40 | 排序按鈕 |
| 透明輪廓 | transparent | Orange | Orange | 離開房間、返回 |
| 禁用狀態 | 同上 + 50% 透明度 | — | — | 非自己回合 |

所有按鈕都套用 Low-Poly 按壓效果（4.3 節）。

#### 按鈕 Hover 變化

一般操作按鈕 hover 時：底色 → 主題色、邊框 → 主題色、文字 → 白色（整體「亮起」效果）。

### 5.3 面板 / 卡片

| 類型 | 底色 | 邊框 | 圓角 | 陰影 |
|------|------|------|------|------|
| 主面板（大廳） | white/90 + backdrop-blur | 2px orange/20，底 8px 右 8px | 12px (xl) | 大陰影 |
| 資訊卡片 | Cream | 2px orange/10，底 3px 右 3px | 12px (xl) | — |
| 規則區塊 | Cream | 2px orange/20，底 4px 右 4px | 12px (xl) | — |
| Modal 遮罩 | black/60 | — | — | — |
| Modal 內容 | white | 2px orange/20，底 8px 右 8px | 16px (2xl) | 大陰影 |

### 5.4 輸入框

- 底色：Cream
- 邊框：2px Orange/30，底 4px 右 4px
- Focus 狀態：邊框色變為 Orange 100%
- 字體：Bold，文字色 Dark
- 無 outline，以邊框色變化取代

### 5.5 徽章 / 標籤

小型圓角矩形，用於顯示角色標記、手牌數量等：

| 類型 | 底色 | 文字色 | 範例 |
|------|------|--------|------|
| 房主標籤 | Yellow | Dark | 「房主」 |
| AI 標籤 | Blue | White | 「AI」 |
| 手牌數 | Pink | White | 「12 張」 |
| 牌堆數 | Yellow | Dark（Black） | 「牌堆: 54」 |

### 5.6 Toast 提示

- 位置：螢幕頂部水平居中，距頂 64px
- 底色：Dark（深咖啡棕）
- 文字色：白色
- 圓角：8px
- 陰影：大陰影
- 動畫：上方淡入，3 秒後淡出

### 5.7 計時器條

- 寬度：100%，高度 6px
- 背景：灰色
- 填充色根據剩餘時間變化：
  - ＞ 40%：Green
  - 20%~40%：Yellow
  - ＜ 20%：Tile Red + 脈動動畫

## 六、背景與裝飾

### 6.1 幾何背景花紋

頁面背景使用四方向交錯的三角形漸層，模擬 Low-Poly 切面效果：

- 花紋色：Peach 40% 透明度
- 花紋尺寸：100 × 100 px 重複
- 底色：Warm（`#FFF0E1`）

效果：看起來像柔和的低面數幾何牆紙，不搶眼但有質感。

### 6.2 毛玻璃效果

面板與 header 使用半透明白底 + `backdrop-blur`：

- 面板：`white/90` + `backdrop-blur-md`
- Header：`white/90` + `backdrop-blur-sm`
- 手牌操作區：`white/95` + `backdrop-blur-md`

## 七、頭像系統

使用 SVG 向量圖繪製的 8 款動物/角色頭像，風格統一：

### 7.1 共通規格

- 畫布：80 × 80 SVG viewBox
- 底圓：半徑 36，填充柔和色系
- 臉部圓：半徑 19~21，填充白/奶油色
- 眼睛：半徑 2.5~3 的圓點，深棕色
- 嘴巴：弧線 stroke，2~3px 寬
- 裝飾：耳朵 / 角 / 帽子等特徵

### 7.2 風格要點

- 全部使用幾何基本形狀（圓、橢圓、三角、弧線）
- 無描邊線條（圖形以填色區分）
- 顏色使用主題色系中的色彩
- 每個頭像帶有 `drop-shadow(1px 2px 0 rgba(92,64,51,0.18))` 的 Low-Poly 風格投影

## 八、圖示系統

### 8.1 共通規格

- 端點/接角：round（圓頭圓角）
- 填充：使用對應主題色 + 12%~30% 透明度底色

## 九、動畫與過渡

### 9.1 過渡動畫

| 動畫 | 效果 | 時長 |
|------|------|------|
| 淡入淡出 | opacity 0→1 + 向下平移 10px | 0.3s ease |
| 按鈕按壓 | 向右下平移 + 邊框消失 | 0.1s |
| 牌面 hover | 向左上偏移 + 陰影加大 | 0.15s（彈性曲線） |

### 9.2 彈性曲線

牌面使用自訂 cubic-bezier：`cubic-bezier(0.34, 1.56, 0.64, 1)`
效果：快速彈出、略微超過目標後回彈，像真實紙牌被手指推動。

### 9.3 Keyframe 動畫

| 動畫名 | 用途 | 效果 |
|--------|------|------|
| drawnPulse | 剛抽到的牌 | 綠色光暈 → 漸弱 → 恢復，2 秒 |
| bounce | 等待中、獲勝頭像 | 上下彈跳，循環 |
| pulse | 計時器危急、提示圖示 | 透明度脈動，循環 |

## 十、響應式設計原則

### 10.1 斷點

| 斷點 | 寬度 | 調整項目 |
|------|------|----------|
| 預設 | < 768px（手機） | 牌面較小、按鈕緊湊、隱藏捲軸 |
| md | ≥ 768px（平板/桌機） | 牌面加大、間距加寬、顯示細捲軸 |

### 10.2 手機優先規則

- 手牌區：橫向可滾動，隱藏捲軸，支援拖曳滾動（cursor: grab）
- 桌面區：flex-wrap 自動換行，觸控優先
- 按鈕：允許橫向溢出滾動，防止按鈕被壓縮
- 禁止頁面縮放（`user-scalable=no`）
- 禁止文字選取（`user-select: none`）

### 10.3 桌機增強

- 手牌區顯示 4px 高的細捲軸（Orange 色滑塊）
- 牌面支援 hover 浮起效果
- 增大間距與字體尺寸

## 十二、套用至其他遊戲的檢查清單

將此風格套用到新遊戲時，依序確認：

- [ ] **色彩**：匯入完整色彩系統（二章），確認背景/面板/按鈕/牌面色彩一致
- [ ] **字體**：載入 Noto Sans TC + Nunito，全站 Bold/Black
- [ ] **立體效果**：所有面板、按鈕、牌面套用偏移陰影 + 底右加粗邊框
- [ ] **按壓回饋**：按鈕 :active 向右下平移 + 邊框消失
- [ ] **牌面狀態**：定義正常/hover/選取/禁用四態的視覺變化
- [ ] **背景花紋**：套用四方向三角形 Low-Poly 花紋
- [ ] **毛玻璃**：header 與操作區使用半透明白底 + backdrop-blur
- [ ] **頭像/圖示**：使用 SVG 幾何風格，統一 drop-shadow
- [ ] **動畫**：牌面使用彈性曲線，按鈕使用 0.1s 過渡
- [ ] **響應式**：手機 → 小牌面 + 隱藏捲軸，桌機 → 大牌面 + hover + 細捲軸
